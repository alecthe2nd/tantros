# `Tantros`

Breathes life into Tantros, the planet Anuke has ceased working on. This mod is meant to faithfully recreate what Tantros could have been, had work not ceased.

There are some restrictions I placed on my self with this project:
* This mod is meant to be an extension of vanilla, and thus must interface cleanly with it.
* This mod must reuse vanilla item and liquid content often, if not always. Thus far this is upheld by the inclusion of only three custom resources: Steam, Redcyst, and Bluecyst. The mod reuses most items and liquids, with copper, lead, sand, oxide, metaglass, silicon, and water already integrated and others planned.

# `Contributing`
Any aid you are willing to provide will at least be considered. Note that most suggestions will fall prey to the restrictions outlined above.

If you want to contribute, be prepared to dive into a rewrite of the block system. It will be unintuitive at first, until you realize that it lets you compose arbitrary blocks from block components.
